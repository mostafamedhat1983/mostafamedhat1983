## 🚀 Featured DevOps Project

### AWS EKS Platform with AI Chatbot  
*June 2025 – December 2025*

A production-grade Kubernetes platform designed and deployed from scratch on AWS, following real-world DevOps,
security, and reliability standards.

**Key Highlights**
- Architected and deployed **85+ AWS resources** across development and production environments.
- Provisioned multi-environment infrastructure (**EKS 1.34, VPC, RDS Multi-AZ**) using **Terraform** with a modular design to ensure scalability and environment parity.
- Implemented **defense-in-depth security**:
  - Falco for runtime threat detection  
  - Trivy for container image vulnerability scanning (0 critical findings)  
  - Non-root containers and least-privilege IAM roles
- Adopted **EKS Pod Identity** for fine-grained IAM access, eliminating OIDC complexity.
- Removed bastion hosts by implementing **AWS Systems Manager Session Manager** for secure access.
- Built CI/CD pipelines using **Jenkins, Helm, and immutable AMIs (Packer)**, reducing deployment time by ~40%.
- Deployed a full **observability stack** (Prometheus & Grafana) enabling zero-downtime deployments.
